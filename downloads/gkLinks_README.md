# gkLinks — マイリンク管理ツール

## 概要
よく使う Web サイトのリンクを、カテゴリ・タグ・メモ付きで一元管理する HTML 単体アプリです。
インストール不要。ブラウザで開くだけで使えます。

## 主な機能
- カテゴリ・パネル（リンクカード）の階層管理
- タグによるフィルタリング・全文検索
- ライトモード / ダークモード切り替え
- ドラッグ&ドロップで並び替え
- スターターセット（Google・SNS・生成AI など）を1クリックで追加
- JSON でのインポート / エクスポート（バックアップ対応）
- Quill リッチテキストメモ帳内蔵（HTML/PDF/TXT 出力）
- お気に入り機能・日付別フィルター・一括で開く機能

## 動作環境
- Windows 10 / 11
- Chrome / Edge / Firefox（最新版推奨）
- インストール不要（HTML ファイルをブラウザで開くだけ）

## 使い方
1. `gkLinks.html` をブラウザにドラッグ&ドロップ（またはダブルクリック）
2. 「スターターセットを追加」で初期リンクを読み込む
3. カテゴリ・パネルを追加してカスタマイズ

## データの保存
データはブラウザのローカルストレージに保存されます。
「データ管理」タブから JSON エクスポートでバックアップできます。
