gkLinks

サーバーも DB も無し、HTML 1ファイルで動く高機能リンク管理アプリ。
IT 開発者・PM の業務効率化を本気で考えた 24 機能を盛り込みました。

  • カテゴリ・パネル階層管理 / タグ / 全文検索（AND/OR/除外）
  • 🤖 AI プロンプトライブラリ（18種・Claude Code / Copilot 連携）
  • 🧩 コードスニペット集（18種・Java/TS/C#/VB.NET/SQL ほか）
  • 📊 ダッシュボード（統計・タグクラウド・ヒートマップ）
  • 📋 デイリースタンドアップ自動生成 / 📅 期限リマインダー
  • ⏱ PR レビュータイマー / 工数記録（CSV 出力）
  • 📝 ノートブック（議事録・ADR・設計書テンプレート）
  • ⌘ Cmd Palette (Ctrl+K) / 🎤 音声検索 / 📷 QR 生成
  • 🎨 8 テーマ切替（Dark Editor / Soft Gold / Paper Premium / Light / Forest / Ocean / Cyber / Sakura）
🏢 社内サーバー設置で全社員配布 gkLinks.html をイントラサーバー (IIS / nginx / Apache / 共有ドライブ) に置けば、全社員が同じ URL から開けます。
ライセンス料・月額・ユーザー数課金: 0円。SaaS でよくある「人数 × 月額」が発生しません。
データは各社員のブラウザ内 (LocalStorage) に個別保存、外部送信なし。共通リンク集は gkLinks_starter.json として一発配布できます。
動作環境 Windows 11 ／ Chrome・Edge（最新版推奨）
ZIP を解凍して HTML をダブルクリックで起動、もしくは社内サーバーに配置して URL アクセス
